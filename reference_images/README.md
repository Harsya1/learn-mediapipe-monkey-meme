# Example Reference Images

Tambahkan gambar pose referensi Anda di folder ini.

## Format yang Didukung
- JPG / JPEG
- PNG
- BMP

## Tips Gambar yang Baik
1. Pose tubuh terlihat jelas
2. Background kontras
3. Pencahayaan baik
4. Resolusi minimal 640x480

## Contoh Penamaan
- `tpose.jpg` - Pose T dengan tangan terbentang
- `wave.png` - Pose melambai
- `peace.jpg` - Pose peace sign
- `superhero.png` - Pose superhero

Setelah menambahkan gambar, jalankan `python main.py` untuk memulai deteksi.
